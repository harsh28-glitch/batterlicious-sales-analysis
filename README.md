# Batterlicious Sales Analysis

## 📊 Project Overview
This project analyzes sales data for Batterlicious using Power BI to uncover trends in revenue, customer behavior, and product performance.

## 🔧 Tools Used
- Power BI
- Excel / Data Source

## 📈 Key Insights
- Identified top-selling products
- Analyzed monthly sales trends
- Found peak sales hours/days
- Compared category performance
